# Spam Detection Model Optimization

## Objective
Optimize a Spam Detection model using GridSearchCV and Cross-Validation to improve classification performance.

## Tools & Libraries
- Python
- Pandas
- Scikit-learn
- TF-IDF Vectorizer
- Multinomial Naive Bayes

## Dataset
SMS Spam Detection Dataset

## Project Workflow
1. Load the dataset
2. Preprocess the text data
3. Convert text into numerical features using TF-IDF
4. Split the dataset into training and testing sets
5. Define hyperparameters
6. Apply GridSearchCV for model optimization
7. Find the best parameters
8. Evaluate the optimized model

## Model Used
Multinomial Naive Bayes

## Hyperparameter Tuning
GridSearchCV with 5-Fold Cross Validation
